IoT物聯網程式設計 - MicroPython/ESP8266

物聯網的英文全名是：Internet of Things，縮寫是IoT，簡單的說，就是萬物連網，所有東西（物體）都可以上網，因為所有東西都連上了網路， 所以，我們可以透過任何連網裝置來遠端控制這些連網的東西、就算遠在天涯海角也一樣可以進行監控。

在人工智慧/機器學習和大數據的時代，資料已經成為一種珍貴的資源，資料收集也是這個新時代的一項必備技能，我們可以使用物聯網透過硬體感測器來自行收集數據， 或直接從網路的 Web 網頁來擷取資料，這就是網路爬蟲(Web Scraping) 。

MicroPython 是 2013 年在 Kickstarter 上募資開始建立的，將 Python 3 濃縮載入硬體微控制器的開源專案， 目前已經能移植於 Micro:bit 和 ESP8266/ESP32 等開發板, 可以讓我們使用Python程式來控制硬體開發板。
